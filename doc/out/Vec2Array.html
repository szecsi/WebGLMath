Class: Vec2Array

Vec2Array

Array of two-element vectors of 32-bit floats. May reflect an GLSL array-of-vec2s uniform variable.
Individual Vec2 elements are available through the at method. Methods are available for optimized bulk processing.

new Vec2Array(size)

Creates an array of vectors.
Parameters:
Name Type Description
size Number | Array The number of Vec2 elements in the array, or an array of 2n elements.
